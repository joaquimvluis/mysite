YOLO

YOLO is a webapp that allows you to search for bucket list items inspiration. It was developed in Ruby on Rails on the final two weeks of Le Wagon's 10-week bootcamp.

The team was composed by 3 elements, and the tasks were evenly distributed between everyone, so we all got to develop both the front and the back end.

First step was a one day design sprint, where we built a UI kit and created a core user journey using Figma. Our core user journey was:

  1. User lands on homepage, can see multiple experiences and can do a search;
  2. The user sees it's search results;
  3. The user chooses one experience and can see it's details;
  4. The user can add that experience to the bucket list, and chooses a data and a penalty in case the task doesn't get completed;
  5. The user can go to the dashboard and see a summary of tasks to be done.

After teachers validation, at the end of the day we presented our core user journey to the class.
